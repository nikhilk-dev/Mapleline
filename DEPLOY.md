# Deployment

## GitHub Pages Setup

1. **Change Pages Source** – Go to Settings → Pages
2. **Build and deployment** → **Source**: Select **Deploy from a branch**
3. **Branch**: Select `gh-pages` / `/(root)`
4. Click **Save**

The workflow will create and push to the `gh-pages` branch automatically on each push to `main`.

Your site will be live at: **https://nikhilk-dev.github.io/Mapleline/**
